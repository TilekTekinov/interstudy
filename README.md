
# interstudy

An information system for enrolling Erasmus students at PEF CULS.

Developed using the TWM methodology and GP libraries.

First, fork this repository. Then clone your fork:

```
git clone git@github.com:<your-github-handle>/interstudy.git
```

To run the project, open a terminal in the directory where you cloned the repository. Then run:

```
. .\dev\bin\activate.ps
```

This activates the environment with all required libraries. After that, you can start the project with:

```
janet bin/demiurge.janet
```
